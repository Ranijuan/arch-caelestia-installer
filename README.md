# 🐚 Instalador Arch Linux + Caelestia

[![Bash](https://img.shields.io/badge/Bash-4EAA25?logo=gnu-bash&logoColor=fff)](instalador_arch.sh)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Script interactivo en Bash para instalar **Arch Linux + Caelestia** de forma guiada y automática.

## 🚀 Inicio Rápido

```bash
# 1. Clonar repositorio
git clone https://github.com/Ranijuan/arch-caelestia-installer.git
cd arch-caelestia-installer

# 2. Hacer ejecutable
chmod +x instalador_arch.sh

# 3. Ejecutar
./instalador_arch.sh

# 4. Seguir las instrucciones en pantalla
```

## ⏱️ Tiempo de Instalación

- **Rápida:** 30-45 minutos
- **Personalizada:** 45-60 minutos

## 📦 ¿Qué instala?

El script ejecuta **8 pasos automatizados:**

1. ✓ Verificación de conexión
2. ✓ Actualización del sistema
3. ✓ Instalación de dependencias
4. ✓ Descarga de Caelestia
5. ✓ Instalación de Caelestia
6. ✓ Configuración (elige DE + idioma)
7. ✓ Paquetes opcionales (Firefox, LibreOffice, VS Code, etc.)
8. ✓ Finalización

## ✅ Requisitos

- ✓ Sistema Arch Linux o WSL en Windows
- ✓ Conexión a internet activa
- ✓ Al menos 15 GB de espacio libre
- ✓ Acceso de administrador (se pedirá sudo)

## 🆘 Troubleshooting

### "Permiso denegado"
```bash
sudo chmod +x instalador_arch.sh
./instalador_arch.sh
```

### "No se encuentra Internet"
Verifica tu conexión:
```bash
ping 8.8.8.8
```

### El proceso se interrumpió
¡Sin problema! Puedes ejecutar nuevamente desde donde se quedó.
