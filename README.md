# 🐧 Arch Linux + Caelestia Installer

**Un instalador 100% automático en un único ejecutable.**

## ⚡ Uso Rápido

```bash
chmod +x arch-caelestia
./arch-caelestia
```

¡La instalación comienza directamente!

---

## 📖 Documentación

Ver la carpeta `resources/` para:
- Documentación completa en español (LEEME.md)
- Guías de ISO (ISO_BUILD_GUIDE.md)
- Scripts auxiliares
- Solución de problemas

---

## 🚀 Características

✓ Instalación 100% automática  
✓ Detección inteligente de hardware  
✓ Particionamiento UEFI/BIOS  
✓ Caelestia incluida  
✓ Un único ejecutable  

---

## 💻 Requisitos

- Linux (Arch/Debian/Ubuntu)
- 5GB espacio libre
- Acceso root/sudo
- Conexión a internet

---

## 🎮 Usos

```bash
./arch-caelestia              # Instalar directamente
./arch-caelestia --iso        # Compilar ISO
./arch-caelestia --help       # Ver ayuda
./arch-caelestia --version    # Ver versión
```
